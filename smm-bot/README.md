# SMM Bot — Telegram → Asana

Бот: кидаешь задачу в Telegram → создаётся задача в Асане (проект «Instagram KG»,
секция «Входящие»), исполнитель — Aisonet. Фото/видео прикрепляются вложением,
ссылка на событие выдёргивается автоматически.

## Что нужно сделать один раз

### 1. Создать бота в Telegram
1. Напиши [@BotFather](https://t.me/BotFather) → `/newbot`
2. Дай имя и username → получишь токен вида `8123456:AAH...`

### 2. Получить токен Asana
1. https://app.asana.com/0/my-apps → **Create new token** (Personal access token)
2. Скопируй токен (показывается один раз)

### 3. Переоформить секции проекта (вручную, 1 минута)
В проекте «Instagram KG» переименуй секции по статусам:
- Планирование → **📥 Входящие**
- Вехи → **🔄 В работе**
- Дальнейшие действия → **👀 На проверке**
- План информационной работы → **✅ Готово**

(GID секции «Входящие» = `1215525237226403` — при переименовании не меняется,
так что в коде уже всё прописано.)

### 4. Деплой на Railway
1. Залей эту папку в репозиторий GitHub
2. https://railway.app → **New Project** → **Deploy from GitHub repo**
3. В разделе **Variables** добавь переменные (см. ниже)
4. Railway сам поднимет `worker` из Procfile

## Переменные окружения (Railway → Variables)

| Переменная | Значение | Обязательна |
|---|---|---|
| `TELEGRAM_TOKEN` | токен от BotFather | да |
| `ASANA_TOKEN` | personal access token Asana | да |
| `ASANA_PROJECT_GID` | `1215525237226401` | уже дефолт |
| `ASANA_SECTION_GID` | `1215525237226403` | уже дефолт |
| `ASANA_ASSIGNEE_GID` | `1213398188813384` (Aisonet) | уже дефолт |
| `ASANA_WORKSPACE_GID` | `1208507351529750` | уже дефолт |
| `ALLOWED_USERS` | твой TG id (узнать: `/start`) | опционально |

`ALLOWED_USERS` — если оставить пустым, писать боту сможет кто угодно, кто его найдёт.
Лучше указать свой ID (бот покажет его по `/start`), можно несколько через запятую.

## Локальный запуск (для теста)

```bash
pip install -r requirements.txt
export TELEGRAM_TOKEN=...
export ASANA_TOKEN=...
python bot.py
```

## Формат сообщения

```
Снять Reels про БИЗНЕС ФЕСТ 2026     ← название задачи
Нужен динамичный монтаж, 15 сек       ← описание
https://ticketon.kg/event/12345       ← ссылка (уйдёт в блок сверху)
```
+ можно приложить фото/видео, в том числе альбомом.

После отправки бот покажет кнопки:
1. **Приоритет** — 🔴 Высокий / 🟡 Средний / 🟢 Низкий → пишется в поле «Приоритет» Асаны
2. **Срок** — Сегодня / Завтра / Через неделю / Без срока → пишется в Due date Асаны

## Как выглядит готовая задача

**Название:** Снять Reels про БИЗНЕС ФЕСТ 2026
**Priority:** Высокий · **Due date:** 18.06.2026
**Описание:**
```
🔗 Ссылка: https://ticketon.kg/event/12345
```
